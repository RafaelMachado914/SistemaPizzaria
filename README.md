<h2>a) Objetivo </h2>
Uma pizzaria deseja um sistema simples para calcular o valor final de um pedido. O usuário 
deverá escolher o tamanho da pizza, o tipo de entrega e os adicionais desejados, e o sistema 
deverá calcular o total da compra.

<h2>b) Regras de negócio</h2>
Valor da pizza conforme o tamanho:</br>
• Pequena: R$ 30,00. </br>
• Média: R$ 45,00. </br>
• Grande: R$ 60,00. </br>
</br>
Valor da entrega: </br>
• Retirada no local: R$ 0,00. </br>
• Entrega em outra região: R$ 15,00. </br>
</br>
Valor dos adicionais: </br>
• Borda recheada: R$ 8,00. </br>
• Queijo extra: R$ 6,00. </br>
• Refrigerante: R$ 7,00. </br>
</br>
Fórmula do total: </br>
• Total = valor da pizza + valor da entrega + soma dos adicionais:
